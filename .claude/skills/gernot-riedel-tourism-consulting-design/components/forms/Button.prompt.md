The standard call-to-action control — use `primary` for the main action, `accent` (orange) sparingly for a single highlighted action, `secondary`/`ghost` for lower emphasis.

```jsx
<Button variant="primary" onClick={save}>Angebot senden</Button>
<Button variant="accent" size="sm">Jetzt starten</Button>
```

Variants: `primary` | `accent` | `secondary` | `ghost`. Sizes: `sm` | `md` | `lg`. Supports `disabled`.
